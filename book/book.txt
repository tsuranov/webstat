Выделение именованных сущностей из неструктурированных текстов (Named Entity Recognition) и другие процедуры обработки. C# конвертируется в Python

https://github.com/igor-shevchenko/rutermextract 
Библиотека для извлечения ключевых слов из текстов на русском языке. Для морфологического анализа используется pymorphy2. Разработка вдохновлена библиотекой topia.termextract, которая делает то же самое с англоязычными текстами.

https://github.com/yandex/tomita-parser/
Томита-парсер создан для извлечения структурированных данных из текста на естественном языке. Вычленение фактов происходит при помощи контекстно-свободных грамматик и словарей ключевых слов. Парсер позволяет писать свои грамматики и добавлять словари для нужного языка.
